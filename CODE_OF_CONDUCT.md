# ميثاق السلوك

يلتزم مجتمع أطلس الذكاء الاصطناعي ببيئة علمية محترمة ومفتوحة للجميع، بصرف
النظر عن الخبرة أو الخلفية أو الهوية أو اللغة أو القدرة.

## السلوك المتوقع

- نقد الفكرة والدليل من دون مهاجمة صاحبها.
- التصريح بدرجة اليقين والتمييز بين الحقيقة والرأي والتنبؤ.
- احترام حقوق المؤلفين ونسب العمل إلى أصحابه.
- تقبل التصحيح وتوثيق سبب القرارات العلمية والتقنية.
- استخدام لغة مهنية خالية من التحرش أو الإهانة أو التمييز.

## السلوك غير المقبول

- التحرش أو التهديد أو نشر معلومات شخصية من دون إذن.
- التلاعب بالمصادر أو اختلاقها أو إخفاء تعارض مصالح جوهري.
- السخرية الشخصية أو تعطيل النقاش والمراجعة عمدا.
- الادعاء بأن المشروع أو مساهميه يؤيدون جهة أو منتجا من دون إذن.

للمشرفين حق تعديل أو رفض أو إزالة المساهمات المخالفة واتخاذ الإجراء المناسب
داخل المستودع. يمكن إبلاغ مالك المستودع بطريقة الاتصال الخاصة الظاهرة في
ملفه على GitHub. وللانتهاكات الواقعة على منصة GitHub يمكن استخدام أدوات
الإبلاغ الرسمية للمنصة.
