# اطلس الذكاء الاصطناعي — مشروع الخرائط التفاعلية

[![Build and test](https://github.com/Zo-Dns/AI-Atlas/actions/workflows/ci.yml/badge.svg)](https://github.com/Zo-Dns/AI-Atlas/actions/workflows/ci.yml)
[![GitHub Pages](https://github.com/Zo-Dns/AI-Atlas/actions/workflows/pages.yml/badge.svg)](https://github.com/Zo-Dns/AI-Atlas/actions/workflows/pages.yml)

خرائط عصبونية تفاعلية تشرح الذكاء الاصطناعي من الداخل، بالعربية الفصحى مع المصطلحات الانجليزية الكاملة، ومختبرات حية تعمل فعلا في المتصفح. الأطلس الأساسي ملف HTML مكتف بذاته، وترافقه مختبرات 3D مستقلة تعمل من ملفات المشروع المحلية أو من GitHub Pages.

> هذا الملف هو السجل الكامل للمشروع: البنية، اعادة البناء، القواعد الصارمة، سجل التدقيق العلمي لكل خريطة، ونمط الاختبار، والعمل المؤجل. اقراه كاملا قبل التعديل.

**الغوص:** انقر عقدة مثل «البيانات» او «تعلم الآلة» او «النماذج اللغوية» او «التعلم العميق» او «الاخلاق والامان» او «التطبيقات» او «الذكاء الاحتمالي» او «محطات تاريخية» في الخريطة الام ← زر «🧭 غص داخل هذه العقدة» ← تتبدل الشاشة الى العالم المتخصص في نفس النافذة، وزر «◂ عودة» يرجعك. عشرة عوالم فرعية تغوص من الخريطة الام؛ تسعة مختبرات حية (عالم التاريخ سردي بلا مختبر). عقدة «الشبكات العصبية» تغوص الى عالم التعلم العميق حيث محتواها الكامل. يضم عالم **تاريخ الذكاء الاصطناعي** (#history) حاليا 80 محطة عبر 9 عصور، ومرجعياته الأساسية نيلسون «The Quest for AI» وAIMA وماكوردوك وكريفييه. اضيف عالما **التعلم المعزز** (Q-Learning) و**الذكاء الرمزي الكلاسيكي** (بحث A*) في 8 يوليو 2026، ثم عالم **الذكاء الاحتمالي** (#prob) ومختبره الحي للشبكة البايزية في 9 يوليو 2026، واضيف مختبر **قرار التطبيق والنشر** لعالم التطبيقات في 9 يوليو 2026 لربط التطبيقات بالكلفة والعتبة والمراقبة.

## 2. اعادة البناء

المجلد مكتف بذاته ويتطلب Node.js 20 أو أحدث. من داخل المجلد:

```bash
npm run build            # يبني القالب ثم ملف الأطلس النهائي
npm test                 # يشغل ملفات الاختبار الـ29 في الجذر والمختبرات المستقلة
npm run serve            # خادم معاينة محلي على http://localhost:8087/
```

يتيح `npm run verify` تنفيذ البناء والاختبارات معا، وهو الأمر نفسه المستخدم في GitHub Actions قبل النشر.

اختبارات صحة رياضيات المختبرات (90 اختبارا، شغّلها بعد كل بناء):

```bash
node lab-math-test.mjs && node llm-lab-test.mjs && node conv-test.mjs && node ml-lab-test.mjs && node data-lab-test.mjs && node ethics-lab-test.mjs && node apps-lab-test.mjs && node classic-lab-test.mjs && node rl-lab-test.mjs && node prob-lab-test.mjs
```

النشر والفتح المحلي: `ai-how-ai-works.html` هو ملف الأطلس النهائي الوحيد، وهو وثيقة HTML كاملة ذات ترويسة UTF-8 وإعداد viewport. يعمل بالفتح المباشر، ومن خادم محلي، وعند النشر.

## 3. بنية المصدر

**محتوى العوالم** (عدّل هنا لتغيير المفاهيم):
- `ai-content.mjs` · `data-content.mjs` · `ml-content.mjs` · `dl-content.mjs` · `llm-content.mjs`
- كل ملف يصدّر: `GROUPS` (المجموعات + الالوان) · `NODES` (العقد) · `JOURNEY` (خطوات الرحلة + `rel`)

**المحرك والمختبرات:**
- `ai-network-template.html` — محرك الخريطة كله (فيزياء Barnes-Hut + رحلة + بطاقات + بحث + غوص) + مختبر الشبكة العصبية والالتفاف. **هذا اضخم ملف مصدر وأهمه.**
- `llm-lab.html/js` · `ml-lab.html/js` · `data-lab.html/js` · `apps-lab.html/js` — مختبرات العوالم الاخرى (HTML + JS منفصلان لكل عالم)

**البناء:**
- `make-atlas.mjs` — يدمج المحرك والمختبرات التسعة في قالب واحد بعوالم متبدلة بالوسم (`#llm` `#dl` `#ml` `#data` `#ethics` `#apps` `#classic` `#rl` `#prob`). يحوي منطق الغوص وروابط `DEEP_LINKS` وتغليف كل مختبر في عالمه.
- `atlas-build.mjs` — يحقن بيانات العوالم، ويضم `XLINKS` (الروابط العابرة داخل كل خريطة) وحارس التشكيل، وينتج الملف النهائي.

**الاختبارات:** `*-test.mjs` — تستخرج رياضيات كل مختبر من الملف المبني نفسه وتتحقق منها ضد حساب يدوي (لا تعيد تنفيذ نسخة، بل تختبر الكود المنشور فعلا).

`archive/` — سكربتات قديمة من قبل الدمج (`ai-build.mjs`, `llm-build.mjs`, `make-llm-template.mjs`, `build-network.mjs`). غير مستخدمة، محفوظة للمرجع فقط.

## 4. القواعد الصارمة (لا تُكسر)

1. **لا تشكيل عربي نهائيا** — لا شدّة (ّ) ولا فتحة/ضمة/كسرة/سكون/تنوين (U+064B–U+0652) ولا الف خنجرية (U+0670). المدة (آ) والهمزات (أ إ ؤ ئ ء) **حروف مسموحة** لا حركات. ⚠️ هذا الخطأ تكرر في كل خريطة تقريبا؛ افحص قبل البناء: `node -e "const s=require('fs').readFileSync('X-content.mjs','utf8'); console.log((s.match(/[ً-ْٰ]/g)||[]).length)"` — يجب ان يطبع 0. للتنظيف: `s.replace(/[ً-ْٰ]/g,'')` (آمن، لا يمس الهمزة/المدة لانها نقاط ترميز منفصلة). **حارسان في البناء:** (أ) `atlas-build.mjs` يفشل لو وجد حركة في اي ملف **محتوى** (`X-content.mjs`)؛ (ب) `make-atlas.mjs` يفشل لو تسربت حركة الى **القالب المدموج كله** (محرك + مختبرات) — اضيف الحارس الثاني في 9 يوليو 2026 بعد اكتشاف 22 حركة قديمة في ملفات مختبرات ml/data/rl لم يكن يفحصها اي حارس (الحارس الاول يفحص المحتوى فقط لا المختبرات).
2. **الدقة العلمية 100%** — كل خريطة تُدقّق بمصادر رسمية عبر workflow متعدد الوكلاء قبل النشر (انظر القسم 5). لا اسماء/تواريخ/تعريفات مخترعة. التبسيط لا يقلب الحقيقة الى خطأ.
3. **المصطلح الانجليزي كامل والأسماء الرسمية لا تدبلج** — في حقل `e` الاسم الانجليزي الكامل مع الاختصار بجانبه: `Reinforcement Learning from Human Feedback (RLHF)` لا `RLHF` وحده. وتبقى أسماء الأنظمة والنماذج والمنتجات والخوارزميات المسماة بصيغتها الرسمية اللاتينية، مثل `AlphaGo` و`AlphaGo Zero` و`ChatGPT` و`Deep Blue`؛ لا تستبدل بتهجئة عربية قد تفقد الهوية أو تربك القارئ. يسمح بالعربية لشرح ماهية الاسم ووظيفته، لا لإعادة تسمية الكيان الرسمي.
4. **رياضيات المختبرات حقيقية ومبرهنة** — كل مختبر يحسب فعلا (شبكة تتدرب، BPE يدرّب، التفاف حقيقي)، وكل رياضياته تُختبر آليا ضد حساب يدوي قبل النشر (القسم 6).
5. **عرض/قراءة فقط** — الخرائط لا تتصل بالانترنت ولا تعدّل اي ملف. كل البناء داخل هذا المجلد فقط.
6. **المصادر اكاديمية علمية موثوقة 100%** — كل حقيقة (تاريخ، اسم، اسناد، رقم) يجب ان تكون مرساة على مصدر اكاديمي علمي موثوق جدا وتُحقَّق مقابله: ورقة محكّمة، كتاب اكاديمي مرجعي (بناشر وسنة)، وثيقة اولية اصلية (النص الاصلي للمقترح/الورقة)، او صفحة رسمية لمؤسسة/جامعة/جائزة. **لا** مدونات ولا منتديات ولا فيديوهات ترفيهية ولا مصادر ثانوية غير موثقة كأساس لحقيقة. المصدر الثانوي (شرح تعليمي) مسموح كرابط تعلّم اضافي فقط، لا كسند لحقيقة. (اضيفت هذه القاعدة صراحة في 9 يوليو 2026 مع عالم «محطات تاريخية».)

## 5. دستور الخريطة الام: الفلسفة ودلالة الاب والروابط (اقر 10 يوليو 2026)

هذا القسم هو المرجع الملزم لاي تعديل بنيوي على الخريطة الام (وقياسا على الحلقات الاولى للعوالم). كتب جوابا على مراجعة اثارت الاسئلة الصحيحة: هل الخريطة تصنيف علمي؟ ماذا يعني الخط من الجذر؟ من يحق له سكن الحلقة الاولى؟

### المادة 1 — هوية الخريطة: اطلس مفاهيمي تربوي، لا تصنيف صوري

الخريطة الام **خريطة مفاهيم تربوية متعددة الاوجه، تستخدم بنية عرض ملاحية، ولا تدعي ان ابوة العرض تمثل دائما احتواء علميا** (concept map بمعنى Novak & Cañas 2008: مفاهيم + روابط مسماة تكوّن قضايا؛ وبوابات استكشاف لعوالم متعمقة) — **وليست** تصنيفا صوريا (taxonomy) ولا انطولوجيا (ontology). هذا وصف دقيق لواقع المجال لا تواضع: **لا يوجد تصنيف قانوني واحد للذكاء الاصطناعي يمكن الامتثال له**، والمراجع الكبرى الثلاثة تتبنى ثلاث بنى مختلفة:

- **ACM CCS 2012** (تصنيف اكبر جمعية لعلوم الحاسوب): يجعل Machine learning **شقيقا** لـArtificial intelligence لا ابنا له (كلاهما تحت Computing methodologies) — قرار فهرسة عملي معلن.
- **Goodfellow وBengio وCourville** (Deep Learning، MIT Press 2016، الشكل 1.4): احتواء متداخل — الذكاء الاصطناعي ⊃ تعلم الآلة ⊃ تعلم التمثيلات ⊃ التعلم العميق.
- **راسل ونورفيغ** (AIMA ط4): تنظيم **بالقدرات** لا بالانواع — حل المسائل، المعرفة والاستدلال، اللايقين، التعلم، التواصل والادراك والفعل.

ثلاثة مراجع قانونية، ثلاث بنى. اذن كل شجرة عرض واحدة اختيار تحريري بالضرورة، والرصانة ليست في ادعاء «الشجرة الصحيحة» بل في **اعلان دلالة كل خط** — وهو ما تلزم به المواد التالية. اقرب الانساب لبنية الحلقة الاولى الحالية: اجزاء AIMA (رمزي/احتمالي/تعلم/تطبيقات) + اوجه سياقية، مع اعلان الاحتواء العلمي الدقيق في البطاقات (المادة 4-ب).

### المادة 2 — دلالة الاب `p`: علاقة عرض، والدلالة العلمية طبقة معلنة فوقها

حقل `p` في ملفات المحتوى هو **displayParent**: يحدد الهندسة الشعاعية والطي والخط الصلب (دالة `compile` في atlas-build.mjs تحوله فهرسا؛ المحرك يرسم منه روابط الشجرة). **لا يستدل من `p` وحده على «نوع من»**. الدلالة العلمية تحملها طبقة انواع معلنة (المادة 3) تعرض للمستخدم في بطاقة العقدة، ولا يغير `p` الا مع صف تدقيق (المادة 6).

### المادة 3 — انواع الروابط الخمسة المعتمدة

كل علاقة (خط شجرة او رابط عابر) تصنف واحدا من ستة انواع، استنادا الى التمييز القياسي في هندسة المعرفة (W3C SKOS يفصل الروابط الهرمية broader عن الترابطية related — وبمعيار SKOS نفسه فان روابط السياق في هذه الخريطة ترابطية لا هرمية، مما يوجب اعلان النوع في البيانات والواجهة لا في التوثيق وحده؛ وWinston وChaffin وHerrmann 1987 لتعدد معاني part-of):

| النوع | المعنى | امثلة في الخريطة الام |
|---|---|---|
| **فرع-من** (is-a) | تخصص علمي داخل الحقل | ml→root، prob→root، rl→ml |
| **جزء-من** (part-of) | مكوّن او مرحلة داخل كل | attention→transformer، pretraining→llm |
| **يعتمد-على** (uses) | تقنية تستند الى اخرى دون احتواء | dl→gpu، rlhf→rl، cutoff→pretraining |
| **يمكّن** (enables) | طبقة تمكين يقوم عليها العلم دون ان تكون فرعا منه | data→root |
| **مظلة-عابرة** (cross-cuts) | تجميع وظيفي يقطع عدة فروع | genai، agents، ethics، robotics |
| **سياق** (context) | تطبيق / تاريخ / حوكمة تحيط بالعلم | apps، history، governance→root |

(اضيف نوعا `enables` في المراجعة الخبرية الثانية 10 يوليو 2026 لفصل «التمكين» عن «السياق» بعد ان لوحظ ان `ctx` كان يحمل المعنيين — تطبيقا لمبدا ان لكل نوع دلالة واحدة.)

المنفذ بصريا: **صلب** = خط الشجرة العرضية، **متقطع** = الروابط العابرة وخطوط المظلات (دالة `isCrossCut` في المحرك ترسم خط جذر المظلة متقطعا وهالة لا نجما)؛ سطر «التصنيف/العلاقة» في البطاقة وسطرا لغة الخطوط في وسيلة الايضاح نفذا 10 يوليو (البند 2 من المادة 7).

**تصحيح دلالة الحلقات (13 يوليو 2026):** فصل التنفيذ بين **المظلة العابرة علميا** (`rt: cross`) و**العقدة المنقولة بصريا للملاحة** (`nav: true` مع `sp` مختلف عن `p`). الحلقات ليست علامة أهمية أو جودة أو مركزية، بل ترميز للمظلة العابرة فقط. لذلك حصرت في: الوكلاء، الروبوتات الذكية، التفاعل بين الإنسان والحاسوب، الأخلاق والأمان، الذكاء التوليدي، الإبداع الحاسوبي، والذكاء العصبي-الرمزي. وصحح تصنيف الرؤية الحاسوبية ومعالجة اللغة الطبيعية والأنظمة متعددة الوكلاء إلى فروع (`rt: is`) بلا حلقات؛ أما اختلاف موضع عرضها عن الأب العلمي فيظهر بوصلة أب عرض متقطعة. وأعلن الوسم الملاحي نفسه في AGI والحوكمة وXAI للسبب البنيوي ذاته. كما صحح الرسم كي لا يجعل كل روابط أبناء المظلة متقطعة: المتقطع هو وصلة أب العرض إلى المظلة أو العقدة المنقولة فقط، بينما تبقى روابط فروعها الداخلية صلبة. يحرس ذلك `mother-map-visual-semantics-test.mjs` بقوائم صريحة وفحوص للمصرّف والقالبين.

**تعديل موثق للمادة 3 (12 يوليو 2026) — تنويع الروابط العابرة نفسها:** كانت الروابط العابرة الـ200 تخزن ازواجا بلا نوع لكل حافة رغم ان نص هذه المادة يوجب تصنيف «كل علاقة (خط شجرة او رابط عابر)» — فجوة اكدتها ايضا مراجعة GPT 5.6 الثالثة (12 يوليو، من خارج المستودع). نفذ التصنيف للـ200 كلها رابطا رابطا بقراءة اسم ووصف طرفي كل رابط من البيانات الحية (لا تخمينا من المفاتيح)، مع تطبيع الاتجاه (المعتمد قبل الركيزة، الاقدم قبل الاحدث). المفردات: الانواع الدستورية الخمسة القابلة للتطبيق (is/part/uses/enables/cross) + اربعة انواع اضيفت تطبيقا لمبدا «لكل نوع دلالة واحدة» اذ كان فرض uses عليها تصنيفا زائفا: **محطة-تاريخية** (`hist`: معلم مؤرخ يؤثر في مسار مفهوم او معلم لاحق — يغطي historically-precedes، ومنه كل سلاسل عالم التاريخ الزمنية)، **معالجة-مشكلة** (`solves`: تقنية تعالج مشكلة معلنة، كـResNet↔تلاشي التدرج وRAG↔الهلوسة)، **نظير-مقابل** (`peer`: تقابل منهجي مفيد بلا اتجاه، كـSARSA↔Q-Learning وMRF↔الشبكة البايزية — يغطي contrasts-with)، **مؤثر-مباشر** (`affects`: تاثير سببي مباشر بلا احتواء ولا معالجة، كلعنة الابعاد↔اقرب الجيران والحرارة↔الهلوسة). النوع يعرض نصا في بطاقتي طرفي كل رابط (قسم «روابط عابرة:» في البطاقة، بصيغتي اتجاه اسميتين محايدتين نحويا — `XL_PHRASE` في المحرك)، والبناء يفشل عند اي رابط عابر بلا نوع معتمد (`XLINK_TYPES` في atlas-build). واضيف سطر ثالث لوسيلة الايضاح يعلن ان مواضع العقد ومسافاتها فيزياء عرض حية لا مقياس علمي — درءا لقراءة القرب المكاني قرابة معرفية (الملاحظة الوجيهة الوحيدة الجديدة في مراجعة 12 يوليو).

### المادة 4 — سياسة الحلقة الاولى (ابناء root)

عضوية الحلقة الاولى **بروز تحريري لجمهور المتعلمين، لا رتبة تصنيفية**. تضم اربعة اصناف معلنة (قائمة مصدرة v1 — توسيعها ممكن بتعديل دستوري موثق بصف تدقيق، لا بالامر الواقع):

- **(أ) فروع علمية**: ml، classic، prob — فرع-من مؤكدة.
- **(ب) تسطيح عرضي لفروع متداخلة بارزة**: nn، dl، llm. الاحتواء العلمي الدقيق (الذكاء الاصطناعي ⊃ تعلم الآلة ⊃ التعلم العميق، وllm صنف نماذج من dl، وnn عائلة نماذج داخل ml) **يعلن نصا في بطاقاتها** بدل دفنها في العمق الشعاعي حيث يفقد اهم مدخل جماهيري قابلية وصوله. (تحكيم المجلس 10 يوليو: كلا التسلسلين قابل للدفاع — ML ← تعلم التمثيلات ← DL عند Goodfellow، او ML ← شبكات ← شبكات عميقة كاحتواء مجموعات؛ المرفوض فقط تسمية nn «فرعا بحثيا» موازيا للحقول.) السابقة المعاكسة مقصودة: عالم التعلم المعزز كامل يسكن تحت ml في موضعه العلمي الصحيح (Sutton & Barto 2018: «النمط الثالث» للتعلم). وسابقة التسطيح العملي عند ACM نفسها: ML بجوار AI لا تحتها.
- **(ج) اوجه سياقية وتمكينية**: apps وhistory (سياق `ctx`)، وdata (تمكين `enabler` — نوع مكرر في المراجعة الثانية اذ البيانات وقود يقوم عليه العلم لا سياق يحيط به). ليست فروعا من علم الذكاء الاصطناعي، وفي ACM CCS هي اشجار عليا **منفصلة اصلا** (Information systems؛ Applied computing؛ Social and professional topics). بقاؤها في الحلقة قرار عرض سليم بشرط اعلان نوعها.
- **(د) مظلات واطر عابرة**: genai مظلة محتوى (`umbrella`)، agents اطار منظم (`framework` — ميز في المراجعة الثانية عن مظلة المحتوى: الوكيل اطار تنظيمي بمعنى AIMA لا تجميع محتوى)، ethics مظلة معيارية-تقنية (`umbrella`). كلها `rt: cross` لغتها البصرية مميزة فعلا.

**قاعدة**: لا تنضم عقدة الى الحلقة الاولى الا بحقلي `nt` (نوع العقدة) و`rt` (نوع العلاقة) في البيانات + صف في جدول المادة 6 بمصدر — والبناء يفرض ذلك آليا (atlas-build يفشل عند حلقة اولى بلا nt/rt او عقدة ام بلا rt).

### المادة 5 — البوابات مستقلة عن الرتبة

«العالم المتعمق» يعلق على عقدته في موضعها العلمي ايا كان عمقها — rl بوابة كاملة من الطبقة الثانية تحت ml. بروز llm في الحلقة الاولى رغم كونه صنف نماذج هو الاستثناء التحريري المعلن (اهم مدخل للجمهور منذ 2022)، وتعويضه العلمي سطر الاحتواء في بطاقته.

### المادة 6 — حدود المراجعة وجدول التدقيق البنيوي

كل جولات التدقيق المسجلة في هذا الملف **آلية موثقة بالمصادر**: مجالس وكلاء متوازية تتحقق من كل ادعاء مقابل مصادر اولية — وهي **ليست مراجعة بشرية مستقلة**، والفرق يبقى معلنا. واجهة المراجعة البشرية هي هذا الجدول: كل نقل بنيوي (تغيير `p` او `g` او انضمام للحلقة الاولى) يقيد صفا ساعة حدوثه.

**سجل المراجعة الخبرية الاولى — 10 يوليو 2026 (اكتملت ونفذت):** راجع المراجع الخارجي (GPT 5.6) صفوف الجدول الـ18 واحدا واحدا: **5 مقبول، 10 مقبول بتعديل، 3 مرفوض العلاقة** (creativity وxai وmultiagent — نمط واحد: فروع من الجذر عرضت تحت اب ملاحي، وحلها الطبيعي حقل sp). طبقت التصحيحات الستة الحاسمة كلها وتعديلات الصياغة (الصفوف موسومة «بعد المراجعة» ادناه)، مع ثلاثة قرارات تنفيذية موثقة من الطرف المنفذ: (1) رابط creativity–genai العابر لم يضف لازدواجه مع خط العرض p القائم؛ (2) رابط multiagent–prob لم يضف لضبط العمق التربوي (اكتفي بـagents وrl)؛ (3) في ethics نفذ البديل الذي اجازه المراجع (مظلة عابرة) بدل الفصل الى عقدتين، معللا بان عالم #ethics الكامل يوحد الشقين على نمط NIST (القسم 8) — والفصل يبقى خيارا مفتوحا. ترقية مبدئية رافقت التنفيذ: المحرك صار يرسم العقد العابرة من نوعها المعلن rt لا من اسم مجموعتها — الدلالة تقود العرض. حكم المراجع بعد التصحيحات: **الحلقة الاولى مقبولة كمراجعة خبرية علمية اولى**؛ يبقى مراجع مستقل ثان للتعددية.

**اغلاق الجولة الاولى (رد المراجع على التنفيذ):** قبل المراجع القرارات التنفيذية الثلاثة (رفض حافة creativity المزدوجة؛ الاكتفاء بروابط multiagent الى agents وrl مع حفظ علاقة الاحتمال في نص العقدة لا في الرسم — نفذ باضافة الالعاب البايزية الى وصف multiagent؛ ومظلة ethics بشرط الا تختزل البطاقة الامان في الاخلاق — الشرط محقق نصا في rn التي تسمي الشقين المعياري والتقني). **تصنيف منهجي ملزم**: ما جرى «مراجعة متعددة النماذج موثقة» (نموذجان لغويان مختلفا التدريب) لا مراجعة اكاديمية بشرية متعددة الاطراف. **الحالة الواقعية للمراجعة (مثبتة بصدق 10 يوليو 2026):** لا يتوفر حاليا مراجع ثالث مستقل. المتاح: مؤلف (Claude) + مراجع واحد من عائلة نماذج اخرى (GPT 5.6). جلسة Claude جديدة **لا تصلح** مراجعا ثانيا لانها من عائلة المؤلف نفسها — ومراجعة المؤلف لعمله ليست استقلالا. لذلك الوصف الامين للحالة النهائية: **مراجعة متعددة النماذج موثقة (مؤلف + مراجع مستقل واحد عابر للعائلات) بسجل خلافات — لا مراجعة بشرية متعددة الاطراف ولا تثليث بمراجع ثالث.** **بروتوكول التعمية معرّف لا مفعّل**: عند توفر مراجع من عائلة ثالثة او بشري تولّد له حزمة نظيفة من البيانات الحية (قيم حية، مبررات محايدة، صفر اثر لاحكام الجولات السابقة) يستلمها وحدها، ثم تقارن الاحكام ويفتح سجل خلافات يحسم بالمصادر. لا يحتفظ بملف معمّى قائم بذاته اذ لا مراجع ثالث بعد.

**سجل المراجعة الخبرية الثانية — 10 يوليو 2026 (نفس المراجع GPT 5.6، غير معمّاة باعترافه، اعاد التقييم من الصفر وفق البيانات والمصادر):** **منهجيا هي اعادة اشتقاق من نفس المراجع لا ضلع استقلال ثالث** — قيمتها في اعادة الفحص المستقل عن احكامه السابقة، لا في التعددية. النتيجة: **10 مقبول، 8 مقبول بتعديل، صفر رفض** (لا صف يستوجب حذفا او رفضا كاملا — تقدم عن الجولة الاولى). طبقت 7 من 8 + اهم اسهام: **توسيع قاموس الانواع** بفئتين صريحتين منعا لتحميل `ctx` و`umbrella` اكثر من طاقتهما — `enabler`/`enables` (data)، و`framework` (agents). التفصيل: (2) classic خفف تاريخه الحاد؛ (6) llm — **الاسم وسلسلة الاحتواء كانا صحيحين في البيانات الحية اصلا** (اختصار في الحزمة فقط)، واضيف تحسين: استبدل الرمز ⊃ بكلمة «داخل» لوضوح المتعلم؛ (8) data → enabler/enables؛ (12) genai «انظمة ونماذج توليد المحتوى والبيانات» تمييزا عن اي نموذج توليدي احصائي؛ (13) agents → framework؛ (14) robotics اعيد اسمها «الروبوتات الذكية» وrt: is→cross وrn يعلن انها علم يتقاطع مع الذكاء لا يندرج كليا تحته؛ (18) multiagent sp: root→agents (الاب الدلالي الانظف). **خلاف موثق واحد على الصف 15 (creativity):** المراجع اقترح rt: is→cross لتعدد تخصصاتها؛ **ابقيتها is** لان الابداع الحسابي فرع بحثي معترف به داخل الذكاء الاصطناعي له مؤتمره وجمعيته (ICCC / جمعية الابداع الحسابي)، وتعدد تخصصاته في مصادر الالهام لا في التصنيف — تماما كمعالجة اللغة الطبيعية فرع من الذكاء يستلهم اللسانيات دون ان يصير «عابرا». وميزتها عن robotics حقيقية: هندسة الروبوت غير-الذكية مكوّن **مقوّم** للروبوتات (فيصح cross)، اما علم النفس **ملهم** للابداع الحسابي لا مكوّن منه (فيبقى is). rn الصف يذكر التقاطع صراحة. حكم المراجع بعد التعديلات: **الحلقة الاولى مقبولة كمراجعة خبرية علمية ثانية قوية**. الحالة الامينة تبقى: مؤلف + مراجع واحد عابر للعائلات (بجولتين الآن)، لا ضلع ثالث مستقل.

**سجل قرارات المجلس — 10 يوليو 2026:** عرض الدستور v1 على مراجع خارجي (GPT 5.6 بصفة استاذ مشارك في المجلس) فمنح **قبولا مشروطا**: اقر الاتجاه (فصل اب العرض عن العلاقة العلمية، انواع الروابط، اعلان حدود المراجعة) وتحفظ على اربع نقاط: صياغة الهوية، وحدود الاستشهاد بـSKOS، وحدة حكم «الشبكات العصبية ليست حلقة تصنيفية»، وانغلاق قائمة الاصناف («لا خامس لها»). **قبلت التحفظات الاربعة وطبقت في هذه النسخة (v1.1)**، ونفذت طبقة الدلالة فعليا في البيانات والواجهة (nt/rt/sp/rn). **الحالة: الدستور ساري كأساس هندسي-علمي اولي — لا يمثل شهادة اعتماد اكاديمي.** تصحيح اصطلاحي من المجلس (10 يوليو): اقرار مراجع واحد للجدول = **مراجعة خبرية** لا «تحكيما اكاديميا»؛ التحكيم يتطلب مراجعين مستقلين متعددين وتسجيل قراراتهم ومصادرهم. حزمة المراجعة الخبرية الاولى (18 صفا، القيم مستخرجة آليا من البيانات المشحونة): `first-ring-review-packet.md`. وتصحيح تغطية: المؤشر دلاليا 198 من 822 عقدة غير جذرية (~24% — الطبقات البنيوية: الام كاملة + حلقات العوالم الاولى)، والـ624 العميقة تؤشر عالما عالما لا دفعة عمياء.

## 6. سجل التدقيق العلمي (ما فُحص وما صُحّح)

كل خريطة مرّت بجولة تدقيق: 3–4 وكلاء متوازين يبحثون في المصادر الرسمية (أوراق arXiv، توثيق scikit-learn/PyTorch، صفحات نوبل/تورينغ الرسمية، نص GDPR). الخلاصة: **صفر خطأ في الحقائق عبر كل الخرائط** — كل الاسماء والتواريخ تأكدت. التصحيحات كانت في دقة التعريفات والترتيب المنهجي، وهي **مطبّقة في ملفات المحتوى**. **تحديث 8 يوليو 2026:** جرت مراجعة شاملة ثانية (مجلس من 5 مدققين خبراء، خريطة لكل خبير) اكدت صفر خطأ جسيم، وطبقت 11 تصحيح دقة ونقص وتموضع (منها GDPR: الموافقة احد ستة اسس لا الوحيد، LeNet «قراءة» لا «فرز»، تقطير AI ينقل من params الى llm، واضيفت عقد نماذج الانتشار وVAE والتكميم).

- **الذكاء الاصطناعي** (الجولة الاكبر): صُحّح عنوان ورقة تورينغ الى «Computing Machinery and Intelligence»، واسمه «آلان» لا «الان»، وكلفة التدريب «عشرات الى مئات ملايين الدولارات»، ووصف RLHF بخطوتيه (نموذج مكافأة ثم تعلم معزز)، ومثال الانتباه (الساعة/الحقيبة كلاهما مؤنث ليعمل الغموض)، ومصطلح دارتموث صيغ 1955 لا اثناء مؤتمر 1956، و«تريليونات» لا «مليارات» تمارين التوقع، وان ChatGPT «اوصل» التقنية للناس لا «اول» من فعل. **أُضيف 16 مفهوما** لسد فجوات (الموجه النظامي، درجة الحرارة، RAG، تعدد الوسائط MLLM، حد المعرفة، قوانين التوسع، الاستدلال، الاختبارات المعيارية، المفتوح/المغلق، التقطير، استخدام الادوات، الفريق الاحمر، حقن الموجهات، الطاقة، AGI، الذكاء التوليدي).
- **التعلم العميق**: 108 ادعاء، صفر خطأ علمي. صُحّحت 6 حركات تشكيل (شدّة تسللت)، ودقّة تاريخ بوابة النسيان في LSTM (نواة 1997، البوابة لاحقا ~1999)، ومفتاح عقدة غير متسق (`medical-note` → `selfdriving`). أُكّد: نوبل الفيزياء 2024 (هوبفيلد وهينتون)، ResNet مايكروسوفت وImageNet 2015، تورينغ 2018 للثلاثي، AlexNet 2012.
- **تعلم الآلة**: 123 ادعاء، صفر خطأ. **خطأ منهجي حقيقي صُحّح:** الرحلة كانت تضع التقييس قبل تقسيم البيانات (= تسرب بيانات، ويناقض درس الخريطة نفسها) → نُقل بعد التقسيم (يُلائم على التدريب وحده). ودقّة: التدريب/الاستدلال مفهوم عام لكل الانواع، الانحدار الخطي يقلل الفروق الراسية لا العمودية، استقلال بايز مشروط بالفئة، اختيار العتبة على التحقق لا الاختبار.

- **تدقيقات الفروع المعمقة بعد الاطلاق (nn، dl، ml، data، llm)** (11 يوليو 2026): [**llm** (الاضخم): شجرة 21 عقدة، سؤال التشذيب الموروث حسم بالابقاء — كل عقدة عالية البروز ولها نظير عمق مثبت (15/15)؛ ثغرة اسناد واحدة (scaling «هي سبب السباق» ← «المحرك التجريبي» موائمة للتصحيح المعتمد سابقا في العالم)؛ والدخيلان اللونيان creativity/tooluse اعيدا الى g:10 مع تدقيق المحرك (rt يتقدم على اسم المجموعة في الرسم العابر) — مجموعة LLM 23←21.] [**data**: 5 فروع = خط انابيب ادنى + خطر مستعرض، كاملة-دنيا (11 مرشح عمق مثبتون في #data)؛ ثغرة اطلاق في labeling («خلف كل نموذج موجه» ← «معظم» + اضافة الوسم الطبيعي من السجلات: نقرات ومشتريات).] بروتوكول ثابت لكل فرع من الخريطة الام: تدقيق نص كل عقدة ضد المصادر + حكم كمال المجموعة وضرورتها (كل مرشح اضافة يثبت وجوده في العمق قبل رفضه) + الاتساق مع العوالم. النتائج: **nn** 10/10 سليمة + ثغرتا دقة (اضافة الانحياز bias لمعادلة العصبون؛ القيمة المطلقة للوزن والاشارة)؛ **dl** 7 فروع + حفيدة، المجموعة كاملة-دنيا (VAE/LSTM/GRU/GNN/ViT مثبتة في #dl) + ثغرتان («مستوى افهم» ← «تمثيلا اعلى تجريدا»؛ اطلاق gpu «كله مدين لها» ← شرط لازم بجانب البيانات)؛ **ml** 7 فروع + 4 احفاد، ثلاثية الانماط قانونية + ثغرة اطلاق في model («تصبح صحيحة» ← «يقل خطؤها») + **اكتشاف فجوة اطلسية اولى: التعلم الذاتي الاشراف (SSL) كان غائبا من الاطلس كله رغم انه النمط الذي يدرب النماذج اللغوية** — عولجت باضافة عقدة selfsupervised الى انواع التعلم في #ml (بجوار شبه الموجه، برابط عابر اليه) وتسمية النمط في عقدة التدريب المسبق بالام (74←75 عقدة في ml، 833←834 اجماليا). النمط العام للثغرات: كلمات اطلاق او صياغة ركيكة، صفر اخطاء حقائق — مؤشر جودة للجولات السابقة.
- **بناء مختبر النموذج اللغوي 3D من الصفر وربطه** (11 يوليو 2026): ثالث مختبرات experiments/ — **محول GPT حقيقي مصغر تدرب على نصوص الاطلس نفسها** (173 الف حرف من ملفات المحتوى الاحد عشر، مفردات 127 حرفا): طبقتان pre-LN × 4 رؤوس × 64 بعدا، راس مربوط بالتمثيلات، GELU-tanh، قناع سببي — 112,320 معاملا، حيرة تحقق 13.42. التدريب numpy خالص بانتشار خلفي يدوي **ببوابة برهان تدرج عددي الزامية** (144 مسبارا، اسوا خطا 5.3×10⁻⁵؛ يجهض التدريب ان فشلت)، AdamW باحماء وجيب تمام، تهيئة متدرجة البقايا. المتصفح **يعيد الحساب كله** (لا تشغيل تسجيلات): انتباه كل راس، softmax، اللوغتات، الاعتيان بالحرارة — ومشهد Three.js يعرض اقواس الانتباه الحية (سماكتها = الوزن المحسوب الآن) وعمود البقايا وتوزيع الرمز التالي، مع لوحة برهان حي من 5 صفوف وراية امانة صريحة («شاهد آليته لا ذكاءه» — الجشع يدور في حلقة تكرار وهذا درس الحرارة). **10 فحوص آلية خضراء** اقساها: المتصفح يعيد انتاج توليد التدريب المرجعي **حرفا حرفا (80 حرفا)** واللوغتات تطابق بـ5×10⁻⁷ (المرجع محسوب من الاوزان المقربة نفسها). علل اصلحت اثناء البناء: ربط وسائط forward الافتراضية وقت التعريف افسد فحص التدرج؛ np.add.at ابطا التدريب ستة اضعاف (استبدل بضرب one-hot: 2088←325ms/خطوة)؛ طباعة عربية اسقطت cp1252 (البيانات سلمت). **الربط**: 4 مداخل — llm والمحول في الام، transformer2 في #llm، وtransformer في #dl.
- **تدقيق مختبر تصنيف الصور 3D (k-NN) وربطه** (11 يوليو 2026): المشروع الثاني في experiments/ — تصنيف ارقام MNIST حقيقية (1/4/7، تقسيم رسمي، 70 تدريب/30 اختبار لكل فئة) بجار اقرب موزون بالمسافة k=5 على 49 ميزة (تجميع متوسط 7×7 + تقييس على التدريب وحده)، وPCA للعرض 3D مثبت على التدريب وحده مع اعلان صريح ان الاسقاط خريطة شرح لا فضاء القرار. **التدقيق**: 4 اختبارات آلية خضراء (احجام التقسيم، اشكال الميزات، **تطابق مصنف المتصفح مع تنبؤات التصدير 90/90**، الدقة 94.44% محققة)؛ سكربت التوليد نظيف (انضباط تسريب حرفي، بذرة ثابتة، تحقق ترويسات MNIST)؛ المراجع اولية (LeCun؛ Cover & Hart 1967؛ Jolliffe). **اصلاحات التدقيق**: دليل الفئات كان بقايا CIFAR (طائرة/قطة/كلب) والالوان صحيحة — صحح الى الرقم 1/4/7؛ README كان يقول 192 ميزة (بقايا CIFAR) → 49؛ dispose لخطوط الجيران؛ توحيد المصطلح (ميزة لا خاصية، 3D)؛ البدء بالعينة 0 كما هي ولو خاطئة (وهي خاطئة فعلا: حقيقة 1/قرار 4 — ودرسها ذهبي: جاران «4» قريبان يغلبان بالتصويت الموزون 1/d² ثلاثة جيران «1» ابعد)؛ وتوثيق اقتران vendor مع مختبر nn بدل ادعاء الاستقلال. **الربط**: زرا غوص جديدان — «التصنيف» في الام و«k-NN» في عالم #ml — بترقية LAB3D_LINKS الى مداخل {تسمية، رابط} وفحص توافر لكل رابط.
- **ربط مختبر 3D بالاطلس** (11 يوليو 2026): زر «🧊 مختبر 3D» على بطاقتي الشبكات العصبية (الام) وMLP (عالم التعلم العميق) يفتح تجربة experiments/nn-3d-simulation في تبويب جديد — **بفحص توافر HEAD**: يظهر فقط حين تكون شجرة المشروع كاملة متاحة (تشغيل محلي/استضافة الشجرة)، ويبقى مخفيا تلقائيا في نسخة الـartifact المنفردة حيث لا وجود للمجلد (لا وعد بما لا يبلغ). مواضع الملفات لم تتغير (مجلد experiments يستقبل مشاريع اخرى). تحقق حي: الزر يظهر على العقدتين بالتسميتين الصحيحتين ويفتح الرابط الصحيح، ويختفي على غيرهما وصفر اخطاء console. (المختبر نفسه: نموذج MNIST حقيقي 784←96←48←10 بدقة ~96.6% تحقق آليا، برهان ذاتي حي، ورقعتا اداء طبقتا — تبديل العينة 42←7.6ms.)
- **جولة دخان الاطلاق الحية (بوابة الاطلاق 2)** (11 يوليو 2026): جولة سلوكية كاملة في متصفح حقيقي عبر ادوات الفحص الآلي: رحلة الخريطة الام **14/14 خطوة** (كل عقدة مرئية بنص ومفاهيم عاملة)؛ **اختبار سلوكي لاصلاح كاميرا الرحلة**: سحب اثناء الخطوة → انحراف صفر (لا رجوع قسري)، وزر التالي يستانف التتبع؛ البحث (نتائج + تعتيم + «لا نتائج» + مسح)؛ الغوص من بطاقة llm بزر الخريطة (تعتيم ← عالم اللغة) والعودة بزر العودة الى الام؛ **العوالم العشرة كلها**: تحميل صحيح (عقد/رحلة/عنوان) وفتح واغلاق مختبر كل عالم (llm 82 عقدة/23 خطوة/13 مشهد، dl 87/27/20، ml 74/15/14، data 71/16/11، ethics 57/16/6، apps 60/10/12، classic 79/13/14، rl 71/12/13، prob 67/11/13، history 80/17/0 سردي بقرار معلن)؛ **صفر اخطاء console** عبر الجولة كلها.
- **الجوال وختم الشفافية (بوابة الاطلاق 3)** (11 يوليو 2026): اضيف **قرص-تكبير باللمس** الى المحرك (تتبع مؤشرات متعددة: بدء القرص بإصبعين يلغي سحب العقدة ويقفل نقرة البطاقة؛ تكبير مرتكز على منتصف الاصبعين بنفس حدود العجلة 0.08–9؛ رفع اصبع واحد يواصل التحريك بلا قفزة؛ pointercancel معالج) — والتحقق سلوكي بأحداث مؤشر صناعية: انفراج 100←300 بكسل اعطى تكبير ×3.00 بالضبط، والنقرة المفردة ما زالت تفتح البطاقة. واضيف **ختم الشفافية** تحت الاحصاءات في كل العوالم: «آخر تحديث علمي: يوليو 2026 · مراجعة متعددة النماذج موثقة» — تاريخه يحدث يدويا مع جولات تدقيق المحتوى لا آليا مع البناء (التاريخ = آخر تحقق علمي لا آخر تصريف). **بهذا اغلقت بوابات الاطلاق الثلاث.**
- **مسح النبرة الشامل (بوابة الاطلاق 1)** (11 يوليو 2026): مسح آلي للتشبيهات في الملفات الاحد عشر (89 مطابقة خام = ~20 زائفة «يمثل/الامثل» + ~56 امثلة تسمية مشروعة + 13 تشبيها حقيقيا). ابقيت 9 معيارية منيرة (جبل الضباب للنزول التدرجي، الطالب الحافظ لفرط التخصيص، تمرين/امتحان للتقسيم...) ورفعت 4 فولكلورية ببدائل تضيف علما: sysprompt (اسبقية النظام على المستخدم + صلة كسر الحماية بدل «موظف الاستقبال»)، openclosed (تمييز فتح الاوزان عن فتح المصدر + تصحيح مثال GPT بعد gpt-oss 2025 بدل «المطعم/الوصفة»)، promptinjection (الجذر المعماري: لا فصل تعليمات/بيانات + تصنيف OWASP الاول بدل «الورقة المدسوسة»)، rag (عبارة «كتاب مفتوح» الراشدة). العوالم العشرة لم تحتج تغييرا. تحقق في الملف المبني: النصوص الاربعة حاضرة والقديمة صفر.

## 7. نمط اختبار المختبرات (البرهان لا الادعاء)

كل مختبر رياضياته تُستخرج من الملف المبني وتُبرهن آليا. نتائج مختارة تثبت انها حقيقية:
- **مختبر الشبكة العصبية**: تدرّج الانتشار الخلفي طابَق المشتقة العددية بخطأ 3.2×10⁻⁹ · الشبكة تعلّمت من دقة 17% الى 93% · رفع الخطر وحده يقلب قرار الكائن من «طعام» الى «اختباء».
- **الالتفاف**: حافة عمودية مع كاشف حواف عمودية → مركز = -1.000 بالضبط (حساب يدوي) · كواشف الحواف مجموعها صفر · التنعيم يحفظ القيمة الموحدة.
- **تعلم الآلة**: الانحدار اللوجستي يفشل في XOR عند **50% بالضبط** (تخمين عشوائي — برهان حد النماذج الخطية) بينما شجرة القرار تنجح 100% · جيني: نقي=0، متوازن=0.5.
- **النماذج اللغوية**: التوزيعات تجمع الى 1 بانحراف 6×10⁻¹⁵ · الحرارة ترفع الانتروبيا (0.06 عند 0.5 مقابل 4.98 عند 2.0) · BPE يفك ترميزه بلا فقدان على 18 نصا · 50/50 توليدا انتهى برمز التوقف.
- **البيانات**: الجدول يبدأ باتساق 67% ودرجة ناقصة، وكل عملية ترفع بُعدها الى 100% بالضبط، والعمليات نقية لا تفسد الاصل.
- **الاخلاق (العدالة والتحيز)**: متطابقة شولديتشوفا تتحقق على اعداد صحيحة الى 3.3×10⁻¹⁶ · لا زوج عتبتين غير تافه يحقق التكافو الديموغرافي وتكافو الاحتمالات وتكافو التنبؤ معا (اقل فجوة قصوى 0.086 ≫ عتبة 0.02) بينما معدلات اساس متساوية او مصنف مثالي يذيبان التعارض · حذف الخاصية الحساسة يبقي نسبة الاثر المتفاوت 0.61→0.69 بينما وكلاء يعيدون بناءها بدقة ~95%، وازالة الوكلاء ترفعها الى ~0.98.
- **التطبيقات (قرار النشر)**: مصفوفة الالتباس والكلفة المتوقعة تختبران ضد حساب يدوي · الكلفة تساوي صيغة FPR/FNR الموزونة بمعدل الاساس · في الدرجات المعايرة عتبة Bayes تقارب عتبة اقل كلفة · مع كلفة غير متناظرة تختلف عتبة اقل كلفة عن عتبة اعلى دقة · الانحراف يجعل العتبة القديمة غير مثلى.

## 8. قرارات تصميمية وعملها المؤجل

- **عدد المجموعات: محرر من قيد الـ8 وموزع حسب البنية العلمية لكل مجال** (تدقيق مصادر اكاديمية، 8 يوليو 2026): الرقم 8 كان قرارا تصميميا (لوحة الالوان + حمل ادراكي 7±2) لا قيدا علميا، وقد رفع. النتيجة **متباينة عمدا** لا قالبا واحدا: **LLM=10** (فصل «بناء نموذج الاساس» عن «ما بعد التدريب»، و«التوجيه» عن «التوسيع»؛ ونقل LoRA الى «ما بعد التدريب» وMoE الى المعمارية — Zhao et al. §4/§5 وMialon et al.)، **DL=10** (فصل المعماريات الى رؤية/تسلسل/توليدية، ونقل MLP الى لبنات الشبكة — Goodfellow وd2l وCS231n)، **Data=9** (فصل التخزين عن الحوكمة والخصوصية — DAMA-DMBOK)، بينما **ML=8** و**AI الام=8** ابقيتا لان ثماني مجموعاتهما تطابق بنية المصادر فعلا (ESL وscikit-learn) — والتقسيم لمجرد التقسيم هو نفس خطأ الشكل معكوسا. وللعالمين الجديدين (8 يوليو): **الاخلاق=8** (مكتسبة لا مفروضة — فحص غير مؤطر رفض انقساماتها المرشحة لان NIST يوحّد المتانة والامن، ونقل «التفسير الميكانيكي» الى الشفافية)، **التطبيقات=9** (فصل «الطب وعلوم الحياة» عن «العلوم الفيزيائية والرياضية» — AI Index 2025). درس امانة: بنيا اول مرة عند 8 بتاطير خفي من صياغة التعليمات (مدى 6-8 + ثماني مجموعات مقترحة)، فاعيد فحصهما بلا سقف فبان ان التطبيقات تريد 9 — القيد يتسلل ما لم يترك العدد نتيجة للبنية لا هدفا. النظام يدعم اي عدد باضافة الوان الى `GROUPS`.
- **طول كل رحلة (JOURNEY) نتيجة علمية لا رقم مؤطر** (8 يوليو 2026؛ حدث توثيق الأم في 13 يوليو): كانت الرحلات كلها نحو 14 خطوة (خمس من سبع عند 14 بالضبط) — اثر قالب لا اشتقاق. اعيدت كل رحلة اشتقاقا من اطار مرجعي مسمى (مجلس 7 خبراء + 3 مدققين خصوميين، تحقق من المصادر الاولية): **Data=16 وML=15** (دورة CRISP-DM مع جدار تسرب: التقسيم قبل اي ملاءمة احصائية)، **DL=23** (بنية Goodfellow وd2l وCS231n: لبنات ثم تدريب ثم تصنيف المعماريات ثم التوليد)، **LLM=23** (دورة حياة Zhao 2023 وداخلها مسار المحول Vaswani 2017)، **Ethics=15** (وظائف نيست: حوكمة/تاطير/قياس/ادارة فوق دورة الحياة)، **AI الام=14** (تغطي المجموعات الرئيسية الاثنتي عشرة مع الجذر والمحول بوصفه جسرا تقنيا، بتبعية مفاهيمية لا خط انابيب)، **Apps=10** (جولة مجالات على قوس القدرات). المدى 10–23، وكل طول هو عدد الحركات القانونية في اطاره لا هدفا مسبقا. **درس بنيوي:** اربع رحلات كانت معيبة بنيويا لا مجرد متساوية الطول — الام والتطبيقات خط انابيب مركب على خريطة استعراضية، وDL وLLM مسار ضيق يغفل معظم مجاله (DL تمريرة صورة عبر شبكة التفافية وحدها، LLM استدلال بلا تدريب). التدقيق الخصومي: صفر مخالفة ترتيب او تسرب، وخطا واقعي وحيد (تاريخ قاعدة AlphaFold: 2022 لا 2020) صحح. حدثت ايضا تسميات ازرار الرحلات الاربع لتطابق محتواها الجديد.
- **التكرار بين الخرائط مقصود ومفيد لا ممنوع**: المفهوم الذي يحتاجه المتعلم في سياقين (مثل الشبكات العصبية في خريطتي ML وDL) يُذكر في كليهما مع رابط غوص، بشرط الا يتناقض التعريفان.
- **خريطة ML تركز على ما قبل الشبكات العصبية** (الخوارزميات الكلاسيكية) لتجنب تكرار خريطة DL، مع عقدتي جسر: «متى الكلاسيكي ومتى العميق» و«الجسر الى التعلم العميق» (تغوص الى `#dl`).
- **عمل مؤجل مقترح**: (1) توسيع الروابط المعرفية بين ML وDL (العصبون = امتداد للانحدار اللوجستي؛ لماذا اتمت الشبكات هندسة الميزات). (2) ✅ **[منجز 8 يوليو 2026] سد فجوات الاكتمال** — اضيفت 10 عقد مصدقة (تحقق 3 وكلاء مقابل اوراق arXiv ومصادر رسمية): **LLM** تقطير النماذج (هينتون 2015)، RLVR، حوسبة وقت الاستدلال (o1-preview 2024)، تقنيات السياق الطويل (RoPE 2021 / الاستيفاء 2023 / FlashAttention 2022)؛ **DL** الانتباه الذاتي (بهداناو 2014)، تعلم النقل، التمثيلات الشعاعية (word2vec 2013 / GloVe 2014)، محول الرؤية ViT (2020)؛ **ML** دالة الخسارة والتحسين؛ **Data** اصدارات البيانات وتوثيقها (صحائف البيانات 2018). واضيف تطبيع الطبقة LayerNorm (با وكيروس وهينتون 2016) — فلم تبق فجوة اكتمال مرصودة. (3) ✅ **[حسم 11 يوليو 2026]** ملاحظة تضخم مجموعة LLM: تدقيق الفرع الكامل حكم بان الحجم **مبرر** — شجرة llm 21 عقدة كلها عالية البروز الجماهيري ولكل منها نظير عمق مثبت في #llm (15/15 فحصا)، فبطاقات الام ملخصات موصولة لا تفاصيل يتيمة. اما الرقم 23 فسببه دخيلان لونيان (creativity وtooluse كانا g:4 وهما خارج الشجرة) اعيدا الى مجموعة العابرة g:10 لمطابقة عائلتيهما العرضيتين، مع تدقيق محرك مرافق: الرسم العابر يتبع rt المعلن اولا واسم المجموعة احتياط لغير المؤشر فقط — فلا يتحول فرع مؤشر (creativity rt:is) الى مظلة بصرية بمجرد لونه. (4) ✅ **[منجز 8 يوليو 2026] مختبر حي للاخلاق (العدالة والتحيز)** بتبويبين: نظرية استحالة العدالة (شولديتشوفا 2017 / كلاينبرغ-ملاينثن-راغافان 2016-2017) عبر مصفوفة التباس ومتطابقة حية، والتمييز بالوكالة (لماذا لا يكفي حذف الخاصية الحساسة) عبر انحدار لوجستي يتدرب حيا؛ 17 اختبارا اخضر، مُدقّق بمجلس تصميم (5 خبراء) وتدقيق خصومي (3 وكلاء). **مختبرات مؤجّلة قيّمها المجلس** (بترتيب القوة): الخصوصية التفاضلية (آلية لابلاس — المرشح الثاني الاقوى، والاصلح للعالم اذ لا يحمل حساسية بيانات العرق)، نموذج المكافاة برادلي-تيري (نواة RLHF قابلة للحساب دون اوزان)، تحيز التمثيل WEAT على متجهات GloVe حقيقية (يتطلب جلب المتجهات وقت البناء)، والتدرجات المتكاملة للتفسير (بضمانة الاكتمال). سبب التاجيل: كلها اقل عائدا-لكل-جهد من التبويبين المبنيين، او تحمل عائقا لوجستيا (جلب متجهات).

## 9. لاضافة عالم سادس (وصفة سريعة)

1. اكتب `X-content.mjs` (GROUPS + NODES + JOURNEY)، وافحص التشكيل (= 0).
2. اطلق workflow تدقيق بالمصادر الرسمية، طبّق التصحيحات.
3. ابنِ مختبرا حيا: `X-lab.html` + `X-lab.js` (بترويسة `/* ===== X LAB MATH: */` و`/*__LAB_DOM__*/` يفصل الرياضيات النقية عن DOM).
4. في `make-atlas.mjs`: اضف قراءة ملفات المختبر، `RAW_X` في bootstrap، كتلة عنوان العالم، `ACTIVE_LAB`، تغليف HTML وJS، رابط `DEEP_LINKS` من عقدة الام، والتحقق النهائي.
5. في `atlas-build.mjs`: استورد المحتوى، اضف `XLINKS_X`، `compile`، حقن `RAW_X`.
6. اكتب `X-lab-test.mjs` يستخرج `X LAB MATH` ويبرهنه. ابنِ وشغّل كل الاختبارات.
