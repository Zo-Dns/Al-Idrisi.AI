# أطلس الذكاء الاصطناعي التفاعلي

[![Build and test](https://github.com/Zo-Dns/AI-Atlas/actions/workflows/ci.yml/badge.svg)](https://github.com/Zo-Dns/AI-Atlas/actions/workflows/ci.yml)
[![GitHub Pages](https://github.com/Zo-Dns/AI-Atlas/actions/workflows/pages.yml/badge.svg)](https://github.com/Zo-Dns/AI-Atlas/actions/workflows/pages.yml)

أطلس عربي مفتوح المصدر يشرح مفاهيم الذكاء الاصطناعي وعلاقاتها بصريا، مع عوالم غوص متخصصة ومختبرات تفاعلية ومكتبة مصادر رسمية وأكاديمية.

**[استكشف الأطلس مباشرة](https://zo-dns.github.io/AI-Atlas/)**

## ماذا يقدم المشروع؟

- خريطة أم تضم 106 مفاهيم رئيسية في الذكاء الاصطناعي.
- عشرة عوالم غوص متخصصة تضم 730 عقدة تعليمية.
- جولة كبرى من 14 خطوة تربط أسس الذكاء الاصطناعي بتطبيقاته ومسؤوليته.
- شرح عربي واضح مع إبقاء المصطلحات والأسماء الرسمية بالإنجليزية.
- رحلات معرفية مرتبة تساعد على فهم كل مجال تدريجيا.
- مختبرات حية ومختبرات 3D لتجربة الخوارزميات والمفاهيم رياضيا.
- مكتبة من 389 مصدرا رسميا وأكاديميا موثقا، مع روابط ومواضع الاستخدام.

تشمل عوالم الغوص: البيانات، تعلم الآلة، التعلم العميق، النماذج اللغوية الكبيرة، الأخلاق والأمان، التطبيقات، الذكاء الرمزي الكلاسيكي، التعلم المعزز، الذكاء الاحتمالي، وتاريخ الذكاء الاصطناعي.

## التشغيل محليا

يتطلب المشروع Node.js 20 أو أحدث:

```bash
git clone https://github.com/Zo-Dns/AI-Atlas.git
cd AI-Atlas
npm run serve
```

ثم افتح:

```text
http://localhost:8087/ai-how-ai-works.html
```

يمكن أيضا فتح `ai-how-ai-works.html` مباشرة، لكن الخادم المحلي هو الخيار الأفضل لتشغيل جميع المختبرات المستقلة.

## البناء والتحقق

```bash
npm run build    # بناء ملف الأطلس النهائي
npm test         # تشغيل ملفات الاختبار الـ29
npm run verify   # البناء والاختبارات وفحص جاهزية الإطلاق
```

تُشغّل عملية التحقق نفسها تلقائيا في GitHub Actions قبل نشر الموقع على GitHub Pages.

## بنية المشروع باختصار

- `*-content.mjs`: مفاهيم العوالم والعلاقات والرحلات.
- `*-lab.html` و`*-lab.js`: المختبرات التفاعلية المدمجة.
- `experiments/`: مختبرات 3D المستقلة.
- `ai-network-template.html`: واجهة ومحرك الخريطة.
- `make-atlas.mjs` و`atlas-build.mjs`: بناء الأطلس النهائي.
- `*-test.mjs`: اختبارات المحتوى والرياضيات والعلاقات والمختبرات.
