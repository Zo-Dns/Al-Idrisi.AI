# سياسة الأمان

## الإصدارات المدعومة

يدعم المشروع أحدث إصدار منشور وأحدث نسخة على الفرع الافتراضي فقط.

## الإبلاغ عن ثغرة

لا تنشر تفاصيل ثغرة قابلة للاستغلال في Issue عام. استخدم خيار **Report a
vulnerability** في تبويب **Security** الخاص بالمستودع بعد تفعيل Private
Vulnerability Reporting. إذا لم يكن الخيار متاحا، تواصل مع مالك المستودع عبر
طريقة الاتصال الخاصة الظاهرة في ملفه على GitHub.

أرفق المسار المتأثر، وخطوات إعادة الإنتاج، والأثر المتوقع، وأي إصلاح مقترح.
سنؤكد استلام البلاغ ونقيّم أثره قبل النشر العام.

الأطلس موقع ثابت بلا خادم خلفي أو حسابات مستخدمين. ومع ذلك نهتم خصوصا
بمشكلات حقن المحتوى، والروابط أو التبعيات المخترقة، وتسريب بيانات غير مقصود،
وسلامة سير عمل GitHub Pages.
