# الخصوصية

لا يجمع Al-Idrisi.AI أسماء أو عناوين بريد أو مدخلات شخصية، ولا يضم
تحليلات استخدام أو إعلانات أو شيفرة تتبع خاصة بالمشروع. الحسابات والتجارب
تعمل داخل متصفح الزائر، ولا يرسل المشروع نتائجها إلى خادم.

تستخدم المختبرات المنشورة ملفات ثابتة من الموقع نفسه. وعند فتح مصدر خارجي
أو تشغيل فيديو مضمّن، يتصل المتصفح بالجهة الخارجية المعنية وتطبق سياساتها؛
تستخدم روابط الفيديو المضمّن نطاق YouTube ذي وضع الخصوصية المحسن
`youtube-nocookie.com`.

عند استضافة الموقع على GitHub Pages تخضع البنية التحتية وسجلاتها لسياسات
GitHub. لا يضيف المشروع ملفات تعريف ارتباط خاصة به ولا يحفظ حالة دائمة في
المتصفح.
